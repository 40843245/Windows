# Event Viewer
## Intro
A viewer to look at infos about events.

## FAQ
### What can be inspected?
1. Event ID
2. Event Source
3. Log
4. Event Type
5. Modified

etc

### How to Open it?
In Windows 11, one can open it through:
### Method 1

Type eventvwr in terminal.

### Method 2 

Type "Event Viewer" in search box on start Menu.
